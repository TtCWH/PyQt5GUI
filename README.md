图像训练及判断界面程序r1.0

使用方法：
1. 加载(load)默认配置(default_config.json),进行初始化。(进行首次训练之后，以后可以加载自己之前训练过保存的配置文件)

2. 浏览并选择(Browse)自己需要训练的图片集，如本例中的ImageSet。需要注意的是，该图片集包括数个文件夹，每个文件夹的名称代表图片的类别，同一类别的图片必须放置在同一个文件夹内。

3. 点击Settings按钮，在弹出的对话框中，对学习参数进行更改，之后点击Apply按钮保存设置，点击OK按钮退出Settings对话框。

4. 点击Start Training按钮进行训练，训练过程中的具体状态会实时显示在下方文本框内。当进度条走到100%后，训练完毕。

5. 切换标签页至Predict页，可以进行测试。选择一张自己需要测试的图片，再点击Sart Prediction即可进行测试。当然，测试所用的图片，其所属的类别必须是训练模型的类别之一。

6. 根据机器给出的结果，选择Wrong或者Right，做一个反馈，帮助机器越学越好。

7. 退出时，可以选择将自己这次训练的配置保存下来，保存格式为json。下次再点开软件，可以加载该配置文件，进行训练或者测试。
